Open [Link](https://stellular-licorice-4d2b24.netlify.app/)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Testing

### Unit Test

```bash
npm run test
# or
yarn test
```

### e2e Test

```bash
npm run test:e2e
# or
yarn test:e2e
```
